# Project Citadel

Project Citadel is a self-hosted homelab built to strengthen my skills in Linux administration, Docker, networking, cybersecurity, automation, and infrastructure management.

The goal is to design and maintain a production-style environment capable of hosting multiple services while documenting every deployment, challenge, and lesson learned.

## Technologies

- Ubuntu Server 24.04
- Docker & Docker Compose
- Caddy
- Nextcloud
- WireGuard
- Cobblemon Minecraft Server
- Jellyfin
- Hermes (Local AI)
- GitHub
- Bash
- Linux

## Goals

- Learn Linux administration
- Practice Docker containerization
- Build secure public-facing services
- Implement VPN access
- Host a personal cloud
- Deploy a modded Minecraft server
- Host local AI models
- Build a professional portfolio
- Document everything
